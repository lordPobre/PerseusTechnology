# Perseus Enterprise 🚀

Plataforma integral de soluciones tecnológicas desarrollada con **Python/Django** y **Tailwind CSS**.

## ✨ Características
- **Marcación Digital:** Sistema biométrico con cumplimiento legal.
- **Consultoría TI:** Gestión de infraestructura, Microsoft 365 e ISO 27001.
- **Desarrollo Web:** Arquitecturas escalables de alto rendimiento.
- **UI/UX:** Interfaz moderna con fondos interactivos (Vanta.js) y diseño Bento Grid.

## 🛠️ Stack Tecnológico
- Django 5.0
- Tailwind CSS
- Alpine.js
- Vanta.js (Three.js / P5.js)

## 📈 SEO & Optimización Estratégica

- La plataforma ha sido auditada para cumplir con los estándares modernos de indexación y carga:
- Rendimiento: Implementación de defer en scripts críticos y conversión de assets a .webp.
- Meta Estratégica: Arquitectura de bloques dinámicos {% block meta_description %} por vista.
- Indexación: Sitemap dinámico y robots.txt optimizado.
- Semántica: Marcado estructurado (JSON-LD) para visibilidad en Google.
